# 16 Adages on Ignorance

A simple, mobile-friendly website featuring 16 powerful adages about ignorance and knowledge. 

**Live Demo:** `https://HAMMAL8.github.io/adages-on-ignorance`  
*Will go live 1-2 mins after you enable GitHub Pages*

## Features

- **16 curated adages** on ignorance, wisdom, and learning
- **Light/Dark mode** with preference saved to browser
- **Search filter** - find quotes by text or author in real time  
- **Copy button** - copy any quote + author to clipboard with one click
- **Download as image** - save any quote as a PNG with watermark
- **Random quote** - highlight and jump to a random adage
- **Share button** - uses native share on mobile, copies link on desktop
- **Responsive design** - works great on phone, tablet, and desktop
- **Zero dependencies** - single HTML file, no build step needed

## Tech Stack

- **HTML5 + CSS3 + Vanilla JS** - no frameworks
- **html2canvas** - for PNG export of quotes
- **GitHub Pages** - free hosting

## Quick Start

1. **Local**: Download `index.html` and open it in any browser
2. **Deploy**: Upload to GitHub and enable Pages under Settings → Pages → Branch: `main`
